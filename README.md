## Начало работы

### Требования

- Node.js (версия 16 или выше)
- Современный веб-браузер с поддержкой [Web Speech API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Speech_API) и [Broadcast Channel API](https://developer.mozilla.org/en-US/docs/Web/API/Broadcast_Channel_API)

### Установка и запуск

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите проект:
   ```bash
   npm run dev
   ```
3. Откройте вкладку браузера по адресу `http://localhost:8080`

### Использование

#### Голосовое управление

Используйте голосовые команды для управления приложением. 

Доступные команды:

- добавь/добавить/создай [круг | треугольник | квадрат | прямоугольник]
- тема [светлая | темная | системная]
- цвет [красный | синий | зеленый | желтый | розовый | оранжевый]
- удалить последний элемент

Примеры:

- добавь круг
- тема светлая
- цвет синий
- удалить последний элемент
